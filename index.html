💡 如何實現網頁「打開即自動載入 12 週 (84 天) 真實資料庫」？

為了免去每次開啟網頁都要手動上傳檔案的繁瑣步驟，本工作台內置了服務端自動載入機制與瀏覽器永久快取 (IndexedDB) 雙核心架構：

  1. 步驟 1：前往 Athena SQL 分頁，複製第 5 條 SQL 查詢並在 AWS Athena 控制台中執行。
  2. 步驟 2：查詢完成後，點擊「下載 CSV」，並將該檔案重新命名為 database.csv。
  3. 步驟 3：將此 database.csv 檔案直接放到專案目錄根路徑下（與 index.html 處於同個目錄）。
  4. 步驟 4：將專案推送（Push）到您的 GitHub 倉庫，Vercel 將自動重新構建部署。

✨ 構建完成後，任何人訪問您的網頁，系統都會在背景自動偵測下載 database.csv，直接解鎖完整 12 週 (84 天) 數據！同時，拖放上傳過的檔案會永久儲存在您的個人瀏覽器中，即使離線也能秒速查詢！
